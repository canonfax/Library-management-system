# Library Management System

## Leírás
Ez a projekt egy TypeScript alapú könyvtári könyvkezelő rendszer, amely támogatja:
- könyvek kezelését
- kölcsönzést és visszavételt
- kategóriák használatát
- generikus kölcsönzési logikát

## Használt technológiák
- TypeScript
- OOP (öröklés, polymorphism)
- Generics
- Interface-ek
- Union és Intersection típusok

## Projekt struktúra
- models: adatmodellek
- services: üzleti logika
- utils: segédfüggvények
- tests: tesztek
